# ch09 · 数字组织者的基本习惯
> 从《打造第二大脑》第9章

**核心观点:** "有条理"不是天生特质,也不靠选对 App,而靠**习惯**。少数轻量习惯把系统维持在
在线、可用的状态,像给车做定期保养,腾出心智空间去做真正的创意工作。

## Frameworks
- **三大维护习惯** — (1) **专案检查表**:启动/收尾专案时走的固定清单;(2) **每周与每月检讨**:
  定期回看工作与生活、决定要不要调整;(3) **及时整理 (Just-in-time)**:用到时顺手整理,不做
  一次性大扫除。把它们当第二大脑的"定期保养日程"。
- **知识飞轮** — 每次启动新专案先查现有文件夹(专案/领域/资源/档案库)有无可复用的模板、大纲、
  旧材料;复用越多,系统越滚越快。
- **边界即专注** — 这些习惯本质是设时间/空间/意图的边界,帮你分清该专注什么、该忽略什么。

## Techniques
- **专案启动清单**:定目标 → 翻各文件夹找相关旧笔记 → 把命中的加粗/摘要 → 挑几个最相关的
  专案/领域/资源夹快速扫一遍(适可而止,别陷进去)。
- **专案收尾清单**:标记完成状态 → 把可复用产物提炼存好 → 相关材料移入档案库。
- 每周留固定时段做轻检讨:清收件箱、把散落笔记归入 PARA、看下周专案。
- 整理靠"just-in-time":真正打开某笔记用时才顺手加工,而非专门腾一天大整理。

## Anti-patterns
- **一次性完美整理** — 指望某天把所有资料一举归置整齐;系统会立刻重新变乱。改成持续的小维护。
- **把"建第二大脑"等于"下载新 App"** — 只换工具不养习惯,系统很快荒废失序。
- **启动检查表时跑偏** — 翻旧文件夹时被有趣但无关的笔记带走;检查阶段要保持克制、维持推进动力。

## Worked examples
- *启动招募第一名员工* → 走启动清单:把"招募"资源夹整个搬进专案夹,30 分钟复习、标出与招人
  最相关的重点,直接作为流程起点。示范"知识飞轮"复用旧积累。
- *像厨师的收纳小习惯* → 调羹放固定位置、刀具用完即擦、食材摆整齐——第二大脑同理,靠一串
  小习惯维持秩序,而非偶尔一次大整顿。
